# Shop Thang - Website Ban Hang

Website ban hang don gian, chay bang HTML + CSS + JavaScript thuan.

## Cach chay

1. Mo thu muc `shop-thang` bang VS Code
2. Cai extension **Live Server** (Ctrl+Shift+X -> tim "Live Server" -> Install)
3. Click phai vao `index.html` -> chon **"Open with Live Server"**

Hoac: double-click vao `index.html` de mo bang trinh duyet.

## Tai khoan chu shop

- Ten dang nhap: `thang`
- Mat khau: `123`

## Ma giam gia

- THANG10 = giam 10%
- SALE20  = giam 20%
- VIP30   = giam 30%
